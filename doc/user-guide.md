# SEOClaw User Guide

Добро пожаловать в SEOClaw! Это ваше визуальное пространство для управления проектами с помощью ИИ.

## Первая настройка
1. Запустите дашборд.
2. Введите ваш API Key.
3. Подключите вашего OpenClaw агента.

## Обзор возможностей
*   **Kanban Board:** Перетаскивание задач между столбцами.
*   **Time Tracking:** Фиксация времени выполнения задач.
*   **Calendar:** Просмотр задач в календаре.
*   **Analytics Dashboard:** Анализ здоровья проектов и производительности команды.
*   **Notifications Bell:** Свежих уведомления о задачах, сроках и изменениях.

## Команды (Voice/Text)
*   "Добавь задачу в проект 'Название' — описание"
*   "Покажи статус проекта 'Название'"
*   "Обнови бюджет проекта на X"
*   "Сколько задач просрочено?"

Для получения полной помощи используйте панель чата `/chat`.
